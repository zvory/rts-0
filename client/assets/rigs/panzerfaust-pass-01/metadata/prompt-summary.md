# Panzerfaust Pass 01

## Style Reference

Use the accepted Rifleman pass 02 and Machine Gunner pass 01 carry sheets as style references:
strict top-down orthographic RTS token art, chunky no-feet infantry silhouette, crisp dark outline,
flat shading, magenta chroma-key background, and readable weapon shapes at small scale.

## Base Carry Generation

Generate one static WWII-era Panzerfaust infantryman carrying the launcher across the body in the
same ready pose family as the Rifleman. The unit wears a rough white-painted winter uniform over
helmet, coat, shoulders, and pack, with muted straps and compact top-down equipment.

## Long Launcher Iteration

Edit the baseline so only the Panzerfaust becomes much larger, longer, and more visually dominant:
an oversized tube spanning farther across the body, with a larger forward warhead and extended rear
end. Keep the soldier, pose, white-painted uniform, and no-feet top-down style.

## Lab Preview Settings

The accepted preview uses the trimmed alpha cutout as a one-frame Panzerfaust strip in
`panzerfaust-long-preview-1`. The lab profile applies `worldScale: 0.03174`, `tintSlot:
team-light`, and target color adjustment `brightness: 69`, `saturation: 83`, `hue: 100`.
